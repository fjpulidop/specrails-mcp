# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |

## Reporting

If you discover a security issue, please report it responsibly. **Do not open a public issue.**

Send an email to **security@specrails.dev** with:

- A description of the issue
- Steps to reproduce (if applicable)
- Any relevant logs or screenshots

## Response Timeline

- **Acknowledgment**: within 72 hours
- **Remediation target**: within 14 days

We appreciate your help in keeping SpecRails MCP secure.
